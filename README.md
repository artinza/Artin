# Artin Zainalpour — Personal Portfolio

Built with pure HTML, CSS, and vanilla JS. No frameworks, no dependencies, no build step.

## Structure

```
/
├── index.html      → Home / Landing
├── work.html       → Portfolio
├── about.html      → About + Manifesto
├── journey.html    → Timeline
├── connect.html    → Contact
├── 404.html        → Not found
└── vercel.json     → Deployment config (clean URLs)
```

## Deploy to Vercel

1. Push this folder to a GitHub repository
2. Go to [vercel.com](https://vercel.com) → New Project → Import your repo
3. Leave all settings as default → Deploy
4. Done. Live in ~30 seconds.

## Deploy to Netlify (alternative)

1. Push to GitHub
2. Go to [netlify.com](https://netlify.com) → Add new site → Import from Git
3. Build command: *(leave empty)*
4. Publish directory: `/` (or the folder name)
5. Deploy

## Local Preview

Just open `index.html` in any browser. No server needed.
